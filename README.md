# Google 翻译托盘工具

## 简介
一个基于 Tauri 2 的 Windows 系统托盘工具，将 Google 翻译包装为桌面应用，
常驻托盘，快捷键唤出，即开即用。

## 功能
- 系统托盘常驻，关闭窗口隐藏到托盘
- 左键点击托盘 / 右键菜单「显示/隐藏」切换窗口
- 全局快捷键 Ctrl+Alt+X 唤出/隐藏
- 单实例运行，防止重复启动
- 自动聚焦翻译输入框

## 使用方法
1. 下载 Release 中的安装包
2. 安装运行，程序自动常驻系统托盘
3. 点击托盘图标或按 Ctrl+Alt+X 打开翻译窗口
4. 关闭窗口自动隐藏到托盘，右键菜单「退出」关闭程序

## 开发
依赖：Rust + Node.js
